## Release: V1 Beta — 2025-06

### What Users Liked

- AI mission generation felt personalized and exciting
- Completing quests with XP popup was satisfying

### What Confused Users

- "Command Center" name didn't match the game feel
- Unclear how to start a second journey

### Biggest Insight

- Users engage more when the first quest is completable within 5 minutes of signup

### Biggest Mistake

- Top navigation felt like a website, not an app

### What We Learned

- Terminology matters — RPG language increases engagement

### What Changes Next

- Bottom navigation (Task 001)
- Rename Command Center → Home (Task 002)
- Add "Start New Journey" CTA on Home
