# Task 006 — PWA

## Objective

Make Level Up installable as a Progressive Web App with offline shell support and a home screen icon.

## Requirements

- Web app manifest (`manifest.json` or `manifest.webmanifest`)
- App name: "Level Up"
- Theme color matching dark theme (`neutral-950`)
- App icons (192px, 512px minimum)
- Service worker for offline shell (app loads even without network for cached pages)
- Install prompt or guidance for "Add to Home Screen"
- Meta tags for mobile viewport and apple-mobile-web-app

## Acceptance Criteria

- [ ] Manifest validates in Lighthouse PWA audit
- [ ] App installable on Android Chrome (Add to Home Screen)
- [ ] App installable on iOS Safari (Add to Home Screen)
- [ ] Custom icon appears on home screen after install
- [ ] App opens in standalone mode (no browser chrome)
- [ ] Offline: app shell loads with graceful offline message for data pages
- [ ] Lighthouse PWA score ≥ 90

## Things NOT to Change

- Core app functionality or routes
- Supabase auth flow
- Database or API routes
- Existing page layouts (PWA is additive)
